# 內湖國小學生手冊暨家長校園生活入口

repo：`nhps-student-handbook`

本專案為「內湖國小學生手冊暨家長校園生活入口」第一版靜態前台 MVP，主要受眾為家長，高年級學生為輔。第一版目標是先整理校園生活常用資訊的入口、分類、範例卡片與資料格式，讓後續內容盤點、處室查核與前台設計能有一致基礎。

## 第一版定位

- 使用 GitHub Pages 與靜態 JSON。
- 不接 Google Sheets、GAS、Cloudflare Worker 或 LINE Bot。
- 不建立後台、不建立自動推播、不處理表單寫入。
- 不處理學生個資、健康個資或個案資料。
- 敏感議題只提供求助與流程導引，不做霸凌、性平認定或法律結論。

## 目前狀態

本 repo 目前為初始化骨架，包含：

- `index.html`：可直接開啟的靜態首頁。
- `data/`：第一版資料格式與安全範例資料。
- `docs/`：企劃摘要、來源盤點、內容規範與 MVP 驗收標準。

## 安全邊界

- 公開頁面只放一般制度、流程與求助方向。
- 不放學生姓名、家長姓名、個人手機或私人聯絡方式。
- 不放事件細節、案件紀錄或可識別個人的描述。
- 不把 token 或 secret 寫入 repo、文件、頁面或紀錄。
- 校網資料需標示來源狀態；舊年度資料或未查核內容不得直接上線。
