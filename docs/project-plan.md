# 專案企劃摘要

## 目標

「內湖國小學生手冊暨家長校園生活入口」第一版目標，是建立一個可被家長快速查找、也讓高年級學生看得懂的校園生活資訊入口。第一版先以靜態頁與 JSON 資料整理常用問題，作為後續內容盤點、處室查核與系統化開發的基礎。

## 受眾

- 家長：主要使用者，查詢請假、放學、接送、健康中心、午餐、活動與求助窗口。
- 高年級學生：輔助使用者，透過短句知道「我可以怎麼做」與「可以找誰」。
- 導師與行政人員：輔助查詢者，協助確認內容、維護資料與統一說明。

## 第一版範圍

- 建立靜態首頁。
- 建立六大入口。
- 建立 6 至 8 筆安全範例卡片。
- 建立資料格式，保留來源追蹤與處室查核欄位。
- 建立內容規範、來源盤點與驗收標準。

## 六大入口

1. 在校作息
2. 請假與出缺席
3. 上放學與接送
4. 健康與午餐
5. 學習與活動
6. 找人幫忙

## 不做事項

- 不做 Google Sheets、GAS、Cloudflare Worker 或 LINE Bot。
- 不建立後台。
- 不建立推播、通報或即時案件處理功能。
- 不建立 secret 或 token。
- 不處理學生個資、健康個資或個案資料。
- 不做霸凌、性平認定教學或法律結論。

## 後續階段

第一版靜態 MVP 完成後，先進行內容盤點與處室查核。待卡片內容穩定後，再評估 GitHub Pages 正式發布、Google Sheets 資料化、GAS 後台、Cloudflare Worker API 與 LINE Rich Menu / LINE Bot。LINE Bot 若導入，也只能查詢既有手冊內容，不處理個案，不生成法律判斷。
